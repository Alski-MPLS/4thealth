# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

**In short:** be respectful, inclusive, and constructive. Harassment, personal attacks,
and discriminatory language are not welcome.

## Reporting

Report violations by opening a [GitHub issue](../../issues) or contacting the maintainer
via the email listed on their GitHub profile. All reports will be handled with discretion.
